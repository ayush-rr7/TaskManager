TaskManager

TaskManager is a simple web application that helps users manage their daily tasks efficiently. Users can add, update, delete, and mark tasks as completed in an organized way.

Features

Add new tasks
Edit/update existing tasks
Delete tasks
Mark tasks as completed
Clean and simple user interface

Technologies Used
Frontend:React
Backend: Node.js, Express.js
Database: MongoDB, Mongoose

Installation & Setup
Clone the repository:
git clone https://github.com/your-username/TaskManager.git


Navigate into the project folder:
cd TaskManager


Install dependencies:
npm install

Start the application:
npm start
